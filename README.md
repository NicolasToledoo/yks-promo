Extensão para cálculo rápido de promoções e descontos.

## ✨ Funcionalidades
- **Modo %:** Preço original + % de desconto → Preço final.
- **Modo Preço:** Preço original + preço desejado → % de desconto.

## Instalação
1. Baixe ou clone este repositório.
2. Acesse `chrome://extensions` ou `edge://extensions`.
3. Ative o **Modo do desenvolvedor** (canto superior direito).
4. Clique em **Carregar sem compactação** e selecione a pasta do projeto.
5. Fixe o ícone na barra de ferramentas.

## Como Usar
1. Abra a extensão pelo ícone.
2. Insira o **Preço Original** (aceita `,` ou `.`).
3. Escolha o modo e preencha o campo correspondente.
4. Clique em **Calcular**.

## Tecnologias
- HTML5 / CSS3
- Vanilla JavaScript
- Chrome Extension Manifest V3
